## Features:

## Install 
> [IMPORTANT NOTE!] You will need [Node](https://nodejs.org/en/download) installed on your system
``` powershell
npm install 
npm build
```

## Usage: 
To run the program you simply run: 
``` Bash
npm start
```
To run the standalone application:
``` Bash
npm run console
```
